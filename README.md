# Мапа спогадів

Цей проект - веб-додаток для створення та перегляду спогадів на карті. Користувачі можуть додавати спогади з геолокацією, зображеннями та описами.

## Технології

- HTML
- CSS
- JavaScript
- Firebase (Firestore, Storage)
- Leaflet.js (для роботи з картами)
